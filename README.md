# DevHive Job Board

A modern and responsive job board application built with React, JavaScript, CSS, and Vite. The platform allows users to browse job opportunities, explore categories, and interact with a professional job board interface.

## Live Demo

https://jobboard-ui.netlify.app/

## Features

* Responsive design for desktop, tablet, and mobile devices
* Modern and professional user interface
* Featured job listings
* Job category browsing
* Job search functionality
* Saved jobs section
* Company showcase section
* Interactive animations
* Dynamic rendering using React components
* Clean and user-friendly navigation

## Technologies Used

* React.js
* JavaScript (ES6+)
* CSS3
* Vite
* HTML5

## Project Structure

* Component-based React architecture
* Local data management using JavaScript modules
* Responsive CSS styling
* Modular project organization
* Dynamic rendering with React

## Project Goal

This project was developed to strengthen frontend development skills by building a realistic job board platform using React, JavaScript, and CSS. It focuses on responsive design, component reusability, state management, and creating modern user interfaces.
